Fórum

Protótipo simplificado de Fórum desenvolvido em Java:

1- Toda a lógica foi desenvolvida por mim, sem utilização de frameworks e seguindo o padrão de projeto MVC.
2- Banco de dados - MySQL - Foram utilizadas somente duas tabelas: Thread e Palavra(blacklist).
3- Telas - HTML, CSS e JavaScript
4- Lógica de ordenação das threads - Foi utilizado uma fila para ordenar a exibição das threads em árvore.
5- Lógica da blacklist - Foi utilizado expressões regulares para cobrir todas as possibilidades.
