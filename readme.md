<h1>Protótipo simplificado de Fórum</h1>

1- Toda a lógica foi desenvolvida por mim, sem utilização de frameworks e seguindo o padrão de projeto MVC. <br />
2- Banco de dados - MySQL - Foram utilizadas somente duas tabelas: Thread e Palavra(blacklist). <br />
3- Telas - HTML, CSS e JavaScript <br />
4- Lógica de ordenação das threads - Foi utilizado uma fila para ordenar a exibição das threads em árvore. <br />
5- Lógica da blacklist - Foi utilizado expressões regulares para cobrir todas as possibilidades. <br />
<br />
Instruções:<br />

1- Na raiz do projeto se encontra o script do banco de dados - forum.sql. <br />
2- A URL de acesso é: localhost:porta/Forum. <br />
3- Na raiz do projeto se encontra o Forum.war para facilitar testes. <br />
